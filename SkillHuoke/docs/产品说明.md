# 产品说明

SkillHuoke 是一个外贸 AI 获客与客户运营系统。它帮助用户把外贸开发流程标准化：

```text
输入产品和目标市场
-> 生成获客工作流
-> 输出客户池
-> 标注联系方式状态
-> AI 评分
-> 生成开发信和 WhatsApp 话术
-> 进入 CRM 跟进
-> 导出表格
```

## 目标用户

- 外贸工厂老板
- 外贸业务员
- SOHO 外贸从业者
- 外贸获客服务商
- 小型外贸团队

## 当前版本定位

当前版本是开源 MVP，用于验证产品方向和演示业务流程。它不需要数据库，也不需要安装复杂依赖，打开网页即可试用。

## 核心价值

- 把找客户从一次性表格变成可运营的流程
- 把客户联系方式分成可信状态，避免假数据
- 把开发信、WhatsApp 话术和 CRM 跟进放在同一页面
- 让非程序员也能用输入指令的方式生成客户开发结果
