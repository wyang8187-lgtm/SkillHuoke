# 开发路线

## 0.1 本地 MVP

- 静态网页
- 输入指令运行工作流
- 客户池表格
- 客户详情
- 开发信和 WhatsApp 话术
- CRM 状态
- CSV 导出

## 0.2 数据管理版

- 本地 JSON 或 SQLite 存储
- 任务历史
- 客户编辑
- 客户去重
- 导入 CSV

## 0.3 AI 接入版

- 接入 OpenAI API
- AI 总结官网
- AI 判断客户类型
- AI 评分
- AI 生成开发信

## 0.4 数据源接入版

- 搜索引擎结果接入
- 官网联系页解析
- LinkedIn 公司页辅助搜索
- 展会和行业目录数据接入

## 1.0 SaaS 版

- Next.js 前端
- 后端 API
- PostgreSQL 数据库
- 登录和团队权限
- CRM 管道
- 邮件草稿
- WhatsApp Business API
- 仪表盘
